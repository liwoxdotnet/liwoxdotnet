## 🚀 Overview

**LiwoxDotNet** is a digital services platform focused on building **high-performance web applications, cloud infrastructure, and automated systems**.

This project powers the official website:

👉 **https://liwox.net**

It is designed to showcase:
- Digital solutions
- Client services
- Portfolio projects
- DevOps-driven systems

---

## 💼 What We Do

### ⚙️ Web Applications
Custom platforms, dashboards, and scalable web solutions built with modern frameworks.

### ☁️ Cloud & DevOps
Infrastructure automation, CI/CD pipelines, and cloud-native deployments using AWS and Cloudflare.

### 🤖 Automation Systems
API integrations, workflow automation, and system orchestration to streamline business processes.

## ⚡ Getting Started

### Prerequisites
- Node.js 18+
- pnpm / npm

### Installation

```bash
git clone https://github.com/liwoxdotnet/liwoxdotnet.git
cd liwoxdotnet

pnpm install
pnpm dev
