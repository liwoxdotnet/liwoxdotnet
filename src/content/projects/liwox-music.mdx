## The Brief

Music content demands fast delivery. Slow-loading audio platforms lose listeners before they hear a single note. The goal was a platform that felt instant — streaming and showcasing original productions without compromise on performance or user experience.

## What We Built

Liwox Music is a lightweight, high-performance web platform built on Astro and deployed on Cloudflare's global CDN. Every design decision was made with media delivery speed in mind.

- **Astro frontend** — zero JavaScript by default, content rendered at the edge
- **Cloudflare CDN** — audio and media assets delivered from the nearest edge node globally
- **Clean showcase UI** — immersive layout designed for music branding and discovery
- **SEO-optimised structure** — metadata, Open Graph, and structured data for every release
- **Scalable media hosting** — architecture designed to grow with catalogue size

## Technical Decisions

**Astro for the frontend.** Music platforms don't need heavy JavaScript frameworks. Astro ships zero JS by default — pages load instantly, even on slower connections. Interactivity is added only where needed, as isolated islands.

**Cloudflare for global media delivery.** Serving audio from a single origin creates latency for international listeners. Cloudflare's CDN caches and delivers assets from edge locations worldwide — the difference is immediate.

## Results

- Sub-second page loads across all regions
- Audio content delivered from Cloudflare's global edge network
- Clean, professional presentation optimised for audience growth
- Scalable architecture ready to expand with new releases and features
